# Global inequalities in ageing and energy policy influence across a century of transformation

This repository contains the organized code, processed figure data, manuscript figure assets, and supplementary documentation for the ageing-energy policy analysis.

The project is organized around four main-text figures:

| Main figure | Result block | Content |
|---|---|---|
| Figure 1 | Result 1 | Long-run temporal expansion, structural breakpoints, phase dynamics, and spatial diffusion of ageing-energy policy attention. |
| Figure 2 | Result 2 | SDG goal-level document production, country contributions, SDG composition over time, and count-impact comparison. |
| Figure 3 | Result 2 | SDG diffusion, concentration, ageing-energy index geography, and OADR-index alignment. |
| Figure 4 | Result 3 | Persistent blind spots in ageing-related energy policy, including keywords, taxonomy composition, country contributions, blind-spot index, and macro-mechanism trends. |

## Repository structure

| Folder | Purpose |
|---|---|
| `scripts/main_figures/` | Current scripts corresponding to the manuscript Figure 1-Figure 4 panels. |
| `scripts/result1/` | Result 1 analysis and Figure 1 quantitative package scripts. |
| `scripts/result2/` | SDG analysis, SDG target-weight, diffusion, and concentration scripts. |
| `scripts/result3/` | Clean taxonomy, topic-share, country analysis, and blind-spot pipeline scripts. |
| `scripts/methods/` | Ageing-energy SDG index construction and validation scripts. |
| `scripts/si/` | Supplementary analysis and SI assembly scripts. |
| `data/compact/` | Compact processed tables supporting the manuscript figures and SI tables. |
| `data/shareable/` | Processed data prepared for public sharing. |
| `figures/main_text/` | Retained figure assets used for manuscript figure reconstruction or checking. |
| `docs/` | Manifests, supplementary documentation, methods notes, and upload guidance. |
| `inputs/` | Placeholder folders for local input placement when rerunning analyses. |
| `outputs/` | Placeholder folder for regenerated outputs. |
