# Data availability note

The repository contains shareable processed data used to generate the figures, compact analytical summaries, and structured figure panel packages. These files are provided under:

- `data/shareable/`
- `data/compact/`

The original Overton export files are not redistributed in this repository. They were used under access conditions that do not permit public re-upload of the raw source exports. For that reason, the repository shares processed analytical tables and figure-ready datasets rather than the original full raw downloads.

In practical terms, this means:

- the code is public
- the figure-supporting processed data are public
- the raw Overton exports are not bundled

Users who have access to the original source data can place those files under `inputs/raw/` and adapt the scripts for full reruns where relevant.
