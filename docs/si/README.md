# Supplementary Information Materials

This folder contains the curated supplementary-information materials for Results 1–3 that were copied from the live project into the GitHub-ready package.

## Contents

- `supplementary_information_results123.md`
  - consolidated SI draft for Results 1–3
- `supplementary_manifest.md`
  - record of which existing project files were used and which old or missing files were not used
- `summaries/`
  - supporting interpretation notes and Step 8–10 summaries
- `tables/`
  - compact SI tables in CSV format for direct inspection and reuse

## Related scripts

The scripts used to assemble these materials are stored in:

- `scripts/si/`

These scripts were organized here for reproducibility and repository clarity. They reflect the saved project outputs and should not be interpreted as newly rerun analyses unless explicitly executed again in a configured environment.
